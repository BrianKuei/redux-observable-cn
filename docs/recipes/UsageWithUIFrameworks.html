Usage with UI Frameworks

Usage with UI Frameworks

redux 和 redux-observable 都是和 UI 框架无关的。你可以选择自己喜欢的，如果你想将来改变 UI 框架，你主要的业务逻辑将会在 redux + redux-observable 所以 你的 UI 框架仅仅是一个展现层的模版，尽可能的简单方便替换。
